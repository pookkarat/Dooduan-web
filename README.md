# Tarot Web — 78 Cards Ready

โครงสร้างไฟล์สำหรับอัปโหลดขึ้น GitHub Pages

## โฟลเดอร์หลัก
- `index.html` หน้าเว็บหลัก
- `style.css` สไตล์เว็บ
- `script.js` ระบบสับไพ่/เลือกไพ่/แสดงผล
- `cards.js` ข้อมูลไพ่ครบ 78 ใบ พร้อม path รูปภาพ
- `assets/cards/` รูปไพ่จัดเป็นชุด

## โครงสร้างรูปไพ่
```
assets/cards/major/
assets/cards/cups/
assets/cards/swords/
assets/cards/wands/
assets/cards/pentacles/
```

อัปโหลดทั้งชุดนี้เข้า repo เดิม `tarot-web` ได้เลย โดยให้ `index.html` อยู่หน้าแรกของ repo และโฟลเดอร์ `assets` อยู่ระดับเดียวกัน
